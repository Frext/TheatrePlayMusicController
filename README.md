# Theatre Play Music Controller

A lightweight, dedicated audio control application designed specifically for live theatre performances. 

This application was originally created for a theatre play organized by a collaborative group of students, including myself, to ensure smooth audio transitions during our live production.

---

## 📝 How to Use

1. **Play/Pause:** Use the primary controls on each individual audio card to trigger track playback or pause mid-scene.
2. **Active Track Highlight**: Currently active audio tracks are highlighted with a distinct red border. 
3. **Volume Balancing:** Adjust individual track volumes on the fly using the horizontal sliders to balance background ambient sounds against spoken dialogue.
4. **Reset Button:** Press the reset button on the left of any track to reset it back to the beginning.
5. **Safety Prompts:** Built-in confirmation dialogs prevent accidental track resets during live performance.
